anagrams
---

Write a method, anagrams, that takes in two strings as arguments. 
The method should return a boolean indicating whether or not the 
strings are anagrams. Anagrams are strings that contain the same characters, but in any order.

JAVA

Source.anagrams("restful", "fluster"); // -> true
Source.anagrams("cats", "tocs"); // -> false
Source.anagrams("monkeyswrite", "newyorktimes"); // -> true
Source.anagrams("elbow", "below"); // -> true
