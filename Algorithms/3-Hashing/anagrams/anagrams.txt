anagrams
---

Write a method, anagrams, that takes in two strings as arguments. 
The method should return a boolean indicating whether or not the 
strings are anagrams. Anagrams are strings that contain the 
same characters, but in any order.

JAVA

anagrams("restful", "fluster"); // -> true
anagrams("cats", "tocs"); // -> false
anagrams("monkeyswrite", "newyorktimes"); // -> true
anagrams("elbow", "below"); // -> true
