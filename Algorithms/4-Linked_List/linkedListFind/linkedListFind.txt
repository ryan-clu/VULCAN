linked list find
---

Write a method, linkedListFind, that takes in the head of 
a linked list and a target value. The method should return 
a boolean indicating whether or not the linked list 
contains the target.



Node<String> a = new Node<>("a");
Node<String> b = new Node<>("b");
Node<String> c = new Node<>("c");
Node<String> d = new Node<>("d");
a.next = b;
b.next = c;
c.next = d;

// a -> b -> c -> d
linkedListFind(a, "c"); // true
linkedListFind(a, "d"); // true
linkedListFind(a, "q"); // false



Node<String> node1 = new Node<>("jason");
Node<String> node2 = new Node<>("leneli");
node1.next = node2;

// jason -> leneli
linkedListFind(node1, "jason"); // true



Node<Integer> node1 = new Node<>(42);

// 42
linkedListFind(node1, 100); // false
